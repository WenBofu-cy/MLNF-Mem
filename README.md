##  项目介绍

后大模型时代，让AI从「工具」走向「可成长智能体」的底层记忆架构。

MLNF-Mem 是全球首创的面向**人形机器人**与**具身智能**的类脑认知系统。它解决了当前AI在**长期记忆**、**行为一致性**、**经验累积**与**个体演化**中的结构性缺陷，为通用智能体（AGI）提供安全、可控、可成长的记忆大脑中枢。

**核心价值**：与大模型天然互补，大模型负责感知推理，MLNF-Mem 负责长期记忆与行为锚定。

##  核心创新点

1. **总控-子漏斗嵌套结构**：统一规则+动态分类，原创类脑架构
2. **宏观自收敛特性**：微观多样，宏观统一，保证机器人行为稳定
3. **类人三驱动机制**：情绪、意义、重复同构人脑记忆
4. **轻量化终身记忆**：严格容量控制，杜绝记忆爆炸
5. **原生学习能力**：可教育、可自主学习、可成长
